🫁 Lung Cancer Detection using Deep Learning

This project focuses on detecting lung cancer from medical images using deep learning models such as VGG16 and ResNet50. The system classifies lung images into different categories to assist early diagnosis.

📌 Repository

Lung Cancer Detection GitHub Repository

🚀 Project Overview

Early detection of lung cancer significantly increases survival chances. This project uses Convolutional Neural Networks (CNNs) and transfer learning models to analyze medical images and predict whether a tumor is present.

The models used:

VGG16
ResNet50

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/s-raj02/Lung_Cancer_Detection.git
cd Lung_Cancer_Detection
2. Create virtual environment
python -m venv venv
3. Activate environment

Windows:

venv\Scripts\activate
4. Install dependencies
pip install -r requirements.txt
▶️ How to Run

If you are using a notebook:

jupyter notebook

If you are running a Python script:

python app.py
